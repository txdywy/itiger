# 🎰 iTiger - Spectacular Web Slot Machine

A visually spectacular, pure-frontend web slot machine game with stunning visual effects!

## ✨ Features

- **5 Themed Levels** - Las Vegas 🎰, Ancient Egypt 🏛️, Maya Temple 🌿, Deep Ocean 🌊, Cosmic Space 🚀
- **Spectacular VFX** - Canvas-based particle system with fire, explosions, golden showers, blizzards, confetti, and starbursts
- **20 Paylines** - Multiple winning combinations across 5×3 reel grid
- **Free Spins** - 3+ scatter symbols trigger 10 free spins with 2× multiplier
- **5 Win Tiers** - Small, Medium, Big, Mega, and Jackpot with escalating visual celebrations
- **Synthesized Audio** - All sound effects generated via Web Audio API (no external files)
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Free Coins** - Add coins anytime, play for fun!
- **Auto-Spin** - Hands-free play mode
- **GSAP Animations** - Smooth reel spinning with satisfying bounce-stop easing

## 🎮 How to Play

1. Select a theme (Vegas, Egypt, Maya, Ocean, Space)
2. Adjust your bet and number of lines
3. Press **SPIN** (or press **SPACE** on keyboard)
4. Match 3+ symbols on a payline to win!
5. Get 3+ ⭐ scatter symbols for free spins

## 🚀 Tech Stack

- Pure HTML5 / CSS3 / JavaScript (ES6 Modules)
- Canvas 2D API for particle effects
- GSAP 3.x for UI animations
- Web Audio API for synthesized sounds
- CSS Custom Properties for theming
- Zero build step - instant deployment
